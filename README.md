# 🖥️ NeonCore - Futuristic UI Interface

A high-fidelity, interactive, and visually stunning web menu built with a modular CSS architecture and a focus on premium aesthetics.

![NeonCore Preview](https://img.icons8.com/isometric/512/futuristic-city.png) *(Preview placeholder)*

## ✨ Key Features

- **🌐 Modular Architecture**: Clean separation of concerns with dedicated CSS modules for Navigation, Hero, HUD, Modals, and Animations.
- **✨ Premium Visuals**: Comprehensive implementation of **Glassmorphism**, neon glow effects, and a dynamic 150-star procedural background.
- **⚡ Advanced Interactivity**:
    - **Magnetic Buttons**: Elements that react to cursor proximity with a physical feel.
    - **Cursor Glow**: A subtle high-tech light effect that follows the mouse.
    - **System Boot Simulation**: A high-fidelity initialization sequence with animated progress bars.
    - **Interactive Terminal**: A functional command-line modal with typing effects.
- **📚 Self-Documenting Nav**: The navigation menu is linked to the project's own source code, explaining the HTML, CSS, and JS structure in real-time via the terminal.
- **📊 Real-time HUD**: A System Status Bar displaying dynamic latency and core metrics simulation.

## 🛠️ Technology Stack

- **HTML5**: Semantic and high-performance structure.
- **Vanilla CSS**: Advanced modern techniques (Variables, Clamp, Grids, Glassmorphism).
- **Vanilla JavaScript**: Optimization-focused logic for all physics and simulations (60FPS stable).

## 📂 Project Structure

```bash
/futuristic-menu
├── css/
│   ├── base.css       # Design tokens and root resets
│   ├── navbar.css     # Glassmorphism navigation logic
│   ├── hero.css       # Glitch text and typography
│   ├── animations.css # Global keyframes (Twinkle, Scan, Glitch)
│   ├── hud.css        # System metrics styling
│   └── modals.css     # Overlay and terminal transitions
├── script.js          # Core logic, star engine, and interactivity
└── index.html         # Main semantic skeleton
```

## 🚀 How to Use

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Hover over the navigation links to see the magnetic effect.
4. Click **Initialize System** for the boot sequence.
5. Click **Architecture** or **System** to explore the project's own documentation in the terminal.
